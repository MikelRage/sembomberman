# sembomberman
BCSH1 - Semestrální práce - Bomberman (Godot, C#)

Základní seznam funkcionalit
* 2D herní pole reprezentované mřížkou (Grid mapa)
* Pevné a zničitelné bloky
* Hráč ovládaný klávesnicí (Pohyb + pokládání bomby)
* Kolizní systém (Hráč nemůže projít zdí)
* Implementace bomb (Časovač výbuchu + exploze ve tvaru kříže + Ničení zničitelných bloků
* Systém životů nebo okamžitá smrt při zásahu explozí
* Nepřítel řízený AI (Náhodný pohyb + kolizní systém + reakce na explozi)
* Úrovně (Rozdílné mapy + rozdílný počet nepřátel + obtížnost nepřátel)
* Ukládání dat do souboru (High score)
* Hlavní menu (Start + load + end)
* Animace pohybu hráče a nepřátel
* Herní stav (Výhra/prohra + restart úrovně)
